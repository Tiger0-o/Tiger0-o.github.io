what is supply chains?

Supply Chains is a strategic tile-placement game that challenges players to create efficient transportation networks between randomly placed resource depots and factories. Navigate challenging terrain including rivers and water bodies while building the most optimal road and bridge systems.

The primary objective is to connect all factories to at least one supply depot using the most efficient transportation system that takes into consideration the environment. Players must strategically place roads and bridges to create supply chains while minimizing resource costs and optimizing pathfinding through difficult terrain.

Built using Python and Pygame, Supply Chains offers engaging gameplay mechanics including road and bridge placement, terrain navigation, and connectivity challenges. Players can cycle through different road and bridge tile types using number keys (1-6) and rotate them with the R key to create the perfect transportation network.

The game features a sophisticated scoring system where players aim to achieve the highest possible score of 9999 by minimizing the number of placed tiles while still achieving full connectivity between all depots and factories. Every placement decision matters in this strategic puzzle game.
